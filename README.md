# NeWork - Социальная сеть для профессионалов

## Описание проекта

NeWork - это Android-приложение, представляющее собой социальную сеть, которая позволяет пользователям создавать посты и события, указывать свои места работы. Приложение разработано в рамках дипломного проекта по профессии "Android Developer".

## Технологии

- **Язык программирования:** Kotlin
- **Минимальная версия Android:** 24 (Android 7.0)
- **Архитектура:** Clean Architecture, MVVM
- **Внедрение зависимостей:** Dagger Hilt
- **Сетевое взаимодействие:** Retrofit, OkHttp
- **Локальное хранение:** Room Database
- **Асинхронность:** Kotlin Coroutines
- **Навигация:** Navigation Component
- **UI:** Material Design, ViewBinding, DataBinding

## Функциональность

### Аутентификация
- Вход в аккаунт
- Регистрация нового пользователя

### Посты
- Просмотр ленты постов
- Создание/редактирование поста
- Выбор упомянутых пользователей
- Лайки постов
- Детальный просмотр поста
- Удаление поста (только для автора)

### События
- Просмотр списка событий
- Создание/редактирование события
- Выбор типа события (онлайн/офлайн)
- Выбор даты проведения
- Выбор спикеров и участников
- Лайки событий
- Детальный просмотр события

### Пользователи
- Просмотр списка пользователей
- Профиль пользователя (стена + работы)
- Свой профиль с возможностью добавления работ
