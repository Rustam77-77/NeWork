## 📱 О проекте

**NeWork** - это мобильное приложение для Android, социальная сеть для профессионалов. Позволяет пользователям создавать посты и события, указывать места работы, находить единомышленников и профессиональные контакты.

Приложение разработано в рамках дипломного проекта по профессии "Android Developer".

## ✨ Функциональность

### 🚀 Основные возможности

- **📋 Лента постов** - просмотр, создание, редактирование и удаление постов
- **📅 Лента событий** - просмотр, создание, редактирование событий с указанием типа (онлайн/офлайн)
- **👥 Пользователи** - список всех пользователей, просмотр профилей
- **❤️ Взаимодействие** - лайки, участие в событиях
- **🔍 Детальный просмотр** - просмотр детальной информации с картой, упоминаниями, спикерами

### 🔐 Авторизация

- Вход по логину/паролю с валидацией полей
- Регистрация с возможностью загрузки аватара
- Проверка на существующего пользователя

### 👤 Профили

- Просмотр профилей других пользователей
- Вкладки "Стена" (посты пользователя) и "Работы" (места работы)
- Свой профиль с возможностью управления работами (добавление, редактирование, удаление)
- Выход из аккаунта

### 📎 Медиа и вложения

- Загрузка изображений для аватара
- Поддержка вложений (изображения, видео, аудио)
- Отображение ссылок с переходом в браузер
- Указание местоположения на карте
